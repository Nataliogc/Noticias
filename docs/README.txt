# Noticias · Hoteles de Calidad en Ciudad Real

**Última actualización:** 19/10/2025 17:07

## Estructura
- `index.html` – Portada con destacado + grid, filtros y paginación
- `app.js` – Carga de `news.json`, filtros (texto/año/etiqueta), destacado y tarjetas
- `styles.css` – Estilos corporativos
- `news.json` – Datos de noticias (título, fecha, texto, url, img, hotel, tags)
- `posts/` – Plantillas de noticia
  - `demo-minimal.html` (A · Minimal)
  - `demo-hero.html` (B · Hero + pie dorado)
  - `demo-galeria.html` (C · Galería)
- `img/` – Logos y placeholders SVG
